# YuGiOh RUSH DUEL-like Card Generator

画像素材を使用して、遊戯王ラッシュデュエル風のオリジナカード画像を作成できるWebツールです。
シングルカードおよびマキシマムカードの作成に対応しています。

## 機能

* **カード生成**: シングルカード、マキシマムカード（3枚合体）の作成
* **画像編集**: アップロードした画像のトリミング、拡大縮小、回転
* **ステータス入力**: 名前（ルビ対応）、属性、レベル、攻撃力/守備力、種族、テキストの編集
* **デザイン調整**:
    * 各種カード枠の色変更（通常、効果、フュージョン、魔法、罠など）
    * 文字色のレアリティ変更（シルバー、ゴールド、レッドなど）
    * 加工エフェクトの適用（ホイル、パラレル、シークレット）
* **保存**: 作成した画像のダウンロード機能

## 使用方法

1.  このリポジトリをクローンまたはダウンロードします。
2.  `index.html` をWebブラウザで開いてください。
